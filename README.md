## schema_sync

Инструмент для сравнения двух схем PostgreSQL и автоматической генерации SQL-команд для приведения целевой схемы к эталонной.

---

## Описание проекта

Этот SQL-скрипт анализирует структуру двух схем (`schema_a` и `schema_b`) и определяет различия между таблицами и колонками.

На основе этих различий генерируются команды `ALTER TABLE`, которые позволяют синхронизировать структуру базы данных.

---

## Возможности

- Поиск общих таблиц между схемами
- Сравнение колонок по имени
- Проверка различий типов данных
- Проверка длины полей (например varchar(100))
- Проверка NULL / NOT NULL ограничений
- Генерация SQL-команд для миграции

---

## Логика работы

1. Определяются общие таблицы в двух схемах  
2. Извлекается структура колонок из `information_schema.columns`  
3. Выполняется сравнение:
   - тип данных
   - длина поля
   - nullable
4. Генерируются SQL-команды:
   - `ADD COLUMN`
   - `ALTER COLUMN TYPE`
   - `SET / DROP NOT NULL`
